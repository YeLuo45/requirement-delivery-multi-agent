# PRD — P-20260624-003: V46-V51 Release Index and PR Draft Automation

## 1. Background
V40-V45 exposed release operations through CLI and API. The next gap is preparation for human review: operators need a compact index across release histories plus PR body/staging suggestions that can be copied without executing git commands.

## 2. Features

### V46 Release index model
- Build a deterministic index of latest release history per proposal.
- Include latest generatedAt, failed gate count, and dirty file count.

### V47 PR draft markdown
- Render a PR body from release ops payload.
- Include failed gates, verification checklist, and manifest summary.

### V48 Stage command suggestions
- Generate `git add -- <path>` command suggestions from commit manifests.
- Do not execute any git command.

### V49 CLI PR mode
- Add `rdma release-ops --pr-draft`.
- Print PR body and staging suggestions.

### V50 JSON includes index
- Include `releaseIndex` and `stageCommands` in JSON output.

### V51 Docs sync
- Document release-ops CLI in README and README.zh-CN.
